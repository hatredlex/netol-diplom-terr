# Terraform code for creating infrastructure in YC

Репозиторий с кодом Terraform для дипломного проекта Netology.

## Состав

- `bootstrap/` — создание начальной конфигурации YC, доступов, бакета для кранения state инфраструктуры.
- `infra/` — создание основной инфраструктуры, кластера Kubernetes в Yandex Managed Service, container-registry.
